# dispenserAuto
O intuito do trabalho no 1º GQ, é criar toda a parte lógica (menu) do Dispenser. Onde o usuário pode ajustar os horários através dos dias da semana, turno e quantidade
de remédios de acordo com o Dispenser escolhido. Para isso, utilizamos: Arduino Mega 2560, LCD 16x2, Potenciômetro, Joystick, RTC (Real Time Clock), jumpers e leds (para
simular o "drop" do remédio.
